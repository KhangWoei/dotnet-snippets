# API Comparison
This is considering the free tier.

## Finnhub
- 60 API calls/minute
- No data range limit
- Access to live data and EOD

## Alpha Vantage
- 25 API calls/day
- No data range limit
- No access to live data, only EOD

## EODHD
- 20 API calls/day 
- 1 year data range
- Access to live data (delayed by 15 minutes), and EOD
